# 仿《造梦西游》2D动作游戏项目 README

## 🎮 项目简介
本项目是基于 **Qt 6.10.2 (MinGW 64-bit)** 框架开发的仿《造梦西游》风格2D横版动作游戏，实现了角色控制、战斗系统、碰撞检测等核心游戏机制，是一个适合Qt游戏开发入门的实践项目。


---

## 🛠️ 环境要求
- **开发工具**：Qt 6.10.2（MinGW 64-bit 编译器）
- **构建工具**：CMake 3.16+
- **IDE推荐**：Qt Creator 10+
- **系统支持**：Windows 10/11（MinGW环境），可适配Linux/macOS（需对应Qt版本）

---

## 🚀 编译与运行步骤
### 1. 环境准备
1.  安装 Qt 6.10.2，勾选 **MinGW 64-bit** 组件
2.  配置 Qt Creator 的 Kits，确保选择 `Desktop Qt 6.10.2 MinGW 64-bit`

### 2. 项目构建
1.  用 Qt Creator 打开项目根目录的 `CMakeLists.txt`
2.  选择对应 MinGW 64-bit 的构建套件
3.  点击「构建项目」（快捷键 `Ctrl+B`），生成可执行文件
4.  点击「运行」（快捷键 `Ctrl+R`）启动游戏

### 3. 命令行构建（可选）
```bash
# 进入项目根目录
mkdir build && cd build
cmake .. -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH=D:/Qt/6.10.2/mingw_64
mingw32-make -j8
./Game.exe  # Windows下运行
```

## 🎯 核心功能模块
| 模块 | 功能说明 |
|------|----------|
| 角色控制 | 支持左右移动、跳跃、普攻/技能攻击等基础操作 |
| 战斗系统 | 实现多段攻击、伤害计算、技能冷却等战斗逻辑 |
| 碰撞检测 | 角色与怪物、技能与怪物、角色与场景的碰撞判定 |
| 游戏循环 | 基于Qt定时器的帧更新逻辑，保证游戏流畅运行 |
| 摄像机跟随 | 画面自动跟随角色移动，适配横版闯关场景 |

## 📝 开发说明
### 代码规范
- 遵循Qt C++开发规范，类名采用大驼峰命名，成员变量以`m_`为前缀
- 头文件与源文件一一对应，功能模块化拆分清晰
- 关键逻辑添加注释，便于后续维护与功能扩展

### 版本控制建议
- 提交代码时，**忽略 `.qtcreator/`、`build/` 目录**，避免提交编译产物
- 推荐 `.gitignore` 配置：
  ```
  # Qt Creator
  .qtcreator/
  # 编译目录
  build/
  *.o
  *.exe
  # CMake
  CMakeFiles/
  cmake_install.cmake
  Makefile
  ```
